# 用 python 實現三統曆

劉歆的三統曆記載在《漢書·律曆志》中，包括統母、紀母、統術、紀術、歲術、世經等部分。統母、紀母是三統曆計算所用的常數，而統術、紀術、歲術則是三統曆計算所用的公式。世經部分則是劉歆根據三統曆排列的上古至西漢的年曆。

三統曆的“術”部分包括**統術**、**紀術**、**歲術**三部分，與這統術、紀術部分相關的又有**統母**、**紀母**兩種常數。統術是用來計算年月日時等的公式，而紀母則是計算五大行星出現時間及運行情況的公式。這裏將統母、紀母、統術、紀術、歲術分爲不同的模塊：

- `tongmu.py`: 統母
- `jimu.py`: 紀母
- `tongshu.py`: 統術
- `jishu.py`: 紀術
- `constants.py`: 計算所用的常數，包括干支表、十二次、二十四節氣、星度、宿度

要使用這些模塊，只需要引入相應模塊就可以了：

```
from constants import calendarConstants as cal
from tongmu import tongmu
from jimu import jimu
import tongshu as ts
import jishu as js
import suishu
```

以往沒有人將世經按劉歆的辦法排成表格過，一般都是隨文驗算。這種做法對於理解劉歆排列曆譜的體例不利。因此，這裏根據世經將年代按照距上元年數排成表格，即 `世經.csv` 中的表格。劉歆的距上元年數有不統一的地方，見我的博客[《漢書·律曆志》筆記（6）：世經](https://ge-chunyu.github.io/posts/2025-12-lvlizhi6/) 中的討論。與此相關的幾個博客文章都是討論三統曆計算的，博客的文字版在`santongli.md`中。
